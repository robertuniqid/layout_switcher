How does this work ?

$('#trigger-content-switch').click(function(event){
     LayoutSwitcher.Trigger('Hello dear user, you can find what you need here', 'The New Content');
});

How do I set this up ?

First of all you need to include jQuery and the LayoutSwitcher File.

You can optionally add jQuery UI for the kickass animations

Done, what's next :) ?

You need to edit within the layout switcher the following files

LayoutSwitcher = {
     ReplacedObjectId : "to-replace-switcher-container",
     // Replace this with the id of the content to be replaced
     ObjectID : "content-switcher-container",
     // This will be the switched content id
}

      If you're more experienced and not working with the ids, take a look at the Init function and you'll see how flexible this is, it can work with anything

Cool done, and now ?

You're all set, start having fun :) !

Can I hook any actions after display ?

In just a few seconds glance !

$('#trigger-content-switch-second').click(function(event){
     LayoutSwitcher.Trigger('Hello dear user, you can find what you need here', 'The New Content', function(){
          alert('You can easily hook up anything here');
     });
});

How many animations do I have available ?

Unlimited, this script can work with ANY animations on the web, you can find a few cool ones here : http://jqueryui.com/effect/

Is it easy to change the animations ?

You can easily change them, just by copy-paste from the jQuery UI website

Can I get the switched content Easily ?

You can get the jQuery object like this :

LayoutSwitcher.GetObject();

And with the jquery object you can easily get the content

LayoutSwitcher.GetObject().html();
